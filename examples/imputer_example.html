Gaussian Imputer:¶

[11]:

imputer = GaussianImputer(model=trained_model, data = X)
imputer.fit(X)

X_missing = X.copy()
mask = np.zeros_like(X_missing, dtype=bool)
mask[1, 1] = True
X_missing[1, 1] = np.nan

X_imputed = imputer.transform(X_missing, mask)
